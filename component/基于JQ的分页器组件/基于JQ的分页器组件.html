实现简单的分页组件，点击可以切换并获取当前页码

参考手册

参数配置:

  • options.container : object

    最外层容器，用于包含组件

  • options.viewSize : Number

    显示的页码数量（不包含前一页、后一页、首页、尾页四个按键），默认为：1

  • options.totalPage : Number

    总共有多少页（设置时必须大于或等于options.viewSize），默认为：100

  • options.currentPage : Number

    选中的页码，可以设置（必须小于或等于options.totalPage），默认为：1

  • options.currentPagePosition : Number

    选中的页码显示的位置，默认为0（即第一个位置）

  • options.labelFirst : String

    首页按钮显示的内容

  • options.labelPrev : String

    上一页按钮显示的内容

  • options.labelNext : String

    下一页按钮显示的内容

  • options.labelLast : String

    尾页按钮显示的内容

组件方法:

  • getCurrentPage

    获取当前页码值，并返回

组件事件:

  • beforeChange

    页码改变前促发

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.函数参数为：改变前的页码

  • afterChange

    页码改变前促发

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.函数参数为：改变后的页码
