实现简单的分页组件，点击可以切换并获取当前页码

参考手册

参数配置:

  • options.container : object

    最外层容器，用于包含组件

  • options.items : Array

    组合框下拉菜单的数据，每项由value和content组成，例如：[{content:"北京",value:"beijing"}]，默认为：[]。

  • options.readonly : Boolean

    输入框是否可以编辑输入，默认为false：可以编辑。

  • options.boxWidth : Number或String

    组合框的宽度，默认100%。

  • options.originIndex : Number

    初始化后默认选中的值的索引，默认0。

  • options.skinClassName : String

    换肤时的类名

组件方法:

  • getselectContent

    获得选中项的content值，若输入框的中的值不在下拉菜单中，则返回输入框中的值。

  • getSelectIndex

    获得选中项的索引值，若输入框的中的值不在下拉菜单中，则返回-1.

  • reset

    将组合框重置

  • disable

    设置组合框为不可用状态。

  • enable

    设置组合框为可用状态

  • setWidth

    设置组合框的宽度

组件事件:

  • beforeshow

    下拉菜单试图展示时触发

    • 参数:

      参数1: {String}

      自定义的事件类型.

  • aftershow

    下拉菜单展示后触发

    • 参数:

      参数1: {String}

      自定义的事件类型.

  • beforeselectitem

    点击下拉菜单某项前触发

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.函数参数为：点击前选择框里面的内容

  • afterselectitem

    点击下拉菜单某项后触发

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.函数参数为：点击后选择框里面的内容

  • beforehide

    列表隐藏前促发

    • 参数:

      参数1: {String}

      自定义的事件类型.

  • afterhide

    列表隐藏后促发

    • 参数:

      参数1: {String}

      自定义的事件类型.
