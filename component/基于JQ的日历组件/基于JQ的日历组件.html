参考手册

参数配置:

  • options.container : object

    最外层容器，一般用于包含选框和日历组件，作为日历定位的参考元素

  • options.dateInput : object

    容器，一般为选框，用于接收所选择的日期

  • options.format : String

    日期格式，例如："yyyy-mm-dd"或者"yyyy/mm/dd"，默认为"yy-mm-dd"。

  • options.weekStart : String

    定义一周的第一天取值，可以为"Mon"或者"Sun"，默认为"Mon"

  • options.yearRange : Array

    设置年份的范围，若无设置，则为当前年份前后十年

  • options.offsetX : Number

    定位时的偏移量，X方向，默认0

  • options.offsetY : Number

    定位时的偏移量，Y方向，默认0

组件方法:

  • getSelectDate

    获取当前选择的日期

  • show

    显示日历插件

  • hide

    隐藏日历插件

组件事件:

  • beforeSelectDate

    选择某个日期之前触发该事件

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.函数参数为：重新选择之前的日期

  • afterSelectDate

    选择日期之后触发该事件

    • 参数:

      参数1: {String}

      自定义的事件类型.

      参数2: {Function}

      促发事件后调用的函数.回调函数参数为：重新选择之后的日期
