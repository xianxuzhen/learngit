简单的无限轮播组件：多张图片并列展示或者一张，通过两边的按钮切换选中的图片，支持自动滚动和循环滚动

参考手册

参数配置:

  • options.container : object

    轮播器父容器

  • options.items : Array

    描述每个滚动项的字符数据: [{img: "./images/0.png",contentText:"contentText0"},{img: "./images/1.png",contentText:"contentText1"},{img: "./images/2.png",contentText:"contentText2"},{img: "./images/3.png",contentText:"contentText3"},{img: "./images/4.png",contentText:"contentText4"},{img: "./images/5.png",contentText:"contentText5"},{img: "./images/6.png",contentText:"contentText6"}]

  • options.viewSize : Number

    描述一页显示多少个滚动项，默认3

  • options.hadBtn : Boolean

    是否有控制按钮

  • options.skinClassName : String

    给选项卡更换样式的类名，默认为null，写法请参考carousel.css文件

  • options.autoplay : Boolean

    是否开启自动滚动

  • options.duration : Number

    执行一次动画花费的时间

  • options.interval : Number

    间隔多久执行一次动画

组件方法:

  • start

    启动滚动

  • stop

    停止滚动

  • getTotalCount

    取得数据项的总数目

    • 返回值:

      返回值: {Number}

      取得数据项的总数目.
